# ecommerce
WebApi 2.0 Exemple With Visua Studio 2013 Update 5 EntityFrameWork 6.1.3
Projeto de exemplo de WebApi 2.0 

Informações sobre o projeto:
	- Backend desenvolvido com WebApi 2.0 no Visual Studio 2013 e EntityFramework.
	- A Solução foi dividida em 3 projetos(Domain-sem dependência, Infra e Api), para fins de distribuições futuras.
	- Você pode baixar o projeto e rodar no Visual Studio 2013. 
  - Você pode testar as consultas/Inserção de dados através com o Fiddler(http://www.telerik.com/download/fiddler/fiddler4) 
    e/ou com a extensão Postman Rest Client para Chrome. Segue em anexo as telas de exemplo bem como um arquivo 
    texto contendo um registro de exemplo no formato json para inserção.
